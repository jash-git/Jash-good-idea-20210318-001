GitZip for github – 快速下載GitHub 項目中的隨意文件和文件夾[Chrome 擴展]

資料來源: http://www.apprcn.com/gitzip-for-github.html

關於GitHub，相信需要用到它的朋友就不會再陌生了。上面有數不清楚的項目，每次更新要么就是命令更新，要么就是上網頁打包項目下載。但有時候可能一次更新只更新某個文件或文件夾，要是再重新整個項目打包下載就太麻煩了。之前我們也有介紹過可以單獨下載某個文件的擴展GitHub Plus，但它並不能下載文件夾，但用了今天介紹的這款GitZip for github你就可以了。

使用方法:
	01.安裝GitZip for github後，打開GitHub上的項目，在需要下載的文件或者文件夾空白處『雙擊鼠標』
	02.這時該文件或文件夾前就會出現一個鉤，表示已經選擇，並且在瀏覽器右下角還會出現一個下載按鈕。
	03.點擊下載按鈕後，GitZip for github會自動向服務器進行請求，將你需要的文件盒文件夾進行下包，下載到瀏覽器的默認下載文件夾處。
	
PS.而如果下載時提示次數用完，則是擴展中作者的API次數用完了。你可以點擊GitZip for github的圖標，按照裡面的提示來申請API。申請後將API粘貼到輸入框中並按回車就可以了，你個人用的話API的限制次數是完全足夠的。
